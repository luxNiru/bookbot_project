# Bookbot

BookBot is my first [Boot.dev](https://www.boot.dev) project!

Best to run this program locally. 
You can swap out any of the text files for any of your own if need be just make sure it is placed within the books directory within the project directory.

To run this project insert this command line prompt into the teriminal replacing the text file directory with what ever you have in your own project:

"python3 main.py books/book_title.txt"
